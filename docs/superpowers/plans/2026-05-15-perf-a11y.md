# Performance & Accessibility Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Eliminate the highest-impact paint, layout, and accessibility issues across the portfolio — achieving Lighthouse Perf ≥ 95 and A11y = 100 without touching business logic or content.

**Architecture:** Seven independent tasks, ordered by risk (lowest first). Each task is self-contained and can be committed alone. Tasks 1–3 are pure CSS/canvas fixes; Tasks 4–7 are accessibility. No new dependencies needed.

**Tech Stack:** Next.js 15 App Router, React 19, Tailwind v4, CSS modules in `app/css/`, Vitest, Playwright + axe-core.

- [ ] **Step 3: Hoist `ctx.font` outside the draw loop and debounce resize**

In `MatrixRain.tsx`, find the `useEffect` body and apply two changes:

**3a — Hoist font:** The `ctx.font` line currently appears twice (pre-warm + inside `frame()`). Remove it from inside the `frame()` function and keep only the one in `resize()`:

```ts
// Inside resize():
ctx!.setTransform(dpr, 0, 0, dpr, 0, 0);
ctx!.font = `${fontSize}px "JetBrains Mono", monospace`; // ← keep here
// ...

// Inside frame() — remove this line entirely:
// ctx!.font = `${fontSize}px "JetBrains Mono", monospace`;  ← delete
```

**3b — Debounce resize:** Replace the raw listener with a debounced version:

```ts
// Before the window.addEventListener call, add:
let resizeTimer: ReturnType<typeof setTimeout> | undefined;
function debouncedResize() {
  clearTimeout(resizeTimer);
  resizeTimer = setTimeout(resize, 100);
}

// Replace:
window.addEventListener('resize', resize, { passive: true });
// With:
window.addEventListener('resize', debouncedResize, { passive: true });

// In all cleanup returns, replace:
window.removeEventListener('resize', resize);
// With:
clearTimeout(resizeTimer);
window.removeEventListener('resize', debouncedResize);
```

```ts
// tests/a11y/axe.spec.ts
import AxeBuilder from '@axe-core/playwright';
import { expect, test } from '@playwright/test';

test.describe('axe-core a11y scan', () => {
  test('homepage has no automatically detectable accessibility violations', async ({ page }) => {
    await page.goto('http://localhost:3000');
    // Wait for client hydration — shell section is the last heavy island
    await page.waitForSelector('.shell__feed');

    const results = await new AxeBuilder({ page })
      .withTags(['wcag2a', 'wcag2aa', 'wcag21aa'])
      .exclude('.crt-vignette, .crt-overlay, .crt-mask, .crt-noise, .crt-flicker, .crt-scan-beam')
      .analyze();

    expect(results.violations).toEqual([]);
  });

  test('contact form error state has no violations', async ({ page }) => {
    await page.goto('http://localhost:3000');
    await page.waitForSelector('form.contact');
    // Trigger validation error by submitting empty form
    await page.click('button[type="submit"]');

    const results = await new AxeBuilder({ page })
      .include('form.contact')
      .withTags(['wcag2a', 'wcag2aa'])
      .analyze();

    expect(results.violations).toEqual([]);
  });
});
```
