# @erikunha-portifolio/stores

Global UI state management using Zustand.

## Purpose

Provides centralized state management with minimal bundle cost for global UI state that would otherwise require props drilling.

## Stores

Currently, this library is set up for future state management needs. Add stores here as needed for global UI state.

## Benefits

- ✅ No props drilling
- ✅ Automatic persistence (when configured)
- ✅ SSR-safe with proper hydration
- ✅ Type-safe API
- ✅ Minimal bundle cost
