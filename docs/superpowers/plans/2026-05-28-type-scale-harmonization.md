# Type Scale Harmonization Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Update three token values in `typography.json`, rebuild the token CSS, fix the R/M/S button touch targets, and verify the Lighthouse `target-size` gate passes.

**Architecture:** Token values change in `design-system/tokens/typography.json`; the generated `design-system/dist/tokens.css` is rebuilt automatically by `pnpm tokens:build`. All CSS consumers reference tokens — no CSS files need editing except the R/M/S button touch-target fix in `DawMixer.module.css`. Visual smoke check via Playwright MCP is required after the build because `lg` and `3xl` changes affect visible section headers and the hero.

**Tech Stack:** Style Dictionary token pipeline (`pnpm tokens:build`), CSS Modules, Lighthouse CI, Playwright MCP

### Task 1: Update token values

**Files:**
- Modify: `design-system/tokens/typography.json`

- [ ] **Step 1: Edit `typography.json` — change three values**

  Open `design-system/tokens/typography.json` and make exactly these three edits:

  ```json
  "ds-text-size-2xs": { "$value": "10px", "$type": "dimension" },
  ```
  ```json
  "ds-text-size-lg": { "$value": "24px", "$type": "dimension" },
  ```
  ```json
  "ds-text-size-3xl": { "$value": "64px", "$type": "dimension" },
  ```

  Leave all other values untouched (`xs` stays 11px, `sm` stays 12px, `base` stays 14px, `md` stays 16px, `xl` stays 32px, `2xl` stays 48px).

- [ ] **Step 2: Rebuild the token CSS**

  Run: `pnpm tokens:build`

  Expected: no errors; `design-system/dist/tokens.css` is updated. The CSS file should contain `--ds-text-size-2xs: 10px`, `--ds-text-size-lg: 24px`, and `--ds-text-size-3xl: 64px`.

  Verify:
  ```bash
  grep -E 'size-2xs|size-lg|size-3xl' design-system/dist/tokens.css
  ```
  Expected output lines:
  ```
  --ds-text-size-2xs: 10px;
  --ds-text-size-lg: 24px;
  --ds-text-size-3xl: 64px;
  ```

- [ ] **Step 3: Run CI gates**

  Run: `pnpm ci:local 2>&1 | tail -10`

  Expected: all gates pass (650 tests, lint, typecheck, content, naming).

- [ ] **Step 4: Commit**

  ```bash
  git add design-system/tokens/typography.json design-system/dist/tokens.css
  git commit -m "feat(design-system): harmonize type scale — 2xs 10px, lg 24px, 3xl 64px"
  ```
